vector

注意
外层vector：存储的是指向内层vector的指针或迭代器（在实现上可能是一个指针），这些指针或迭代器是连续存储的。
内层vector：每个内层的 vector 都有自己的内存块，用于存储其元素，这些元素在其内部是连续存储的。但当把所有内层vector视为一个整体时，由于它们可能在不同的时间点被创建或重新分配内存（比如当某个内层vector需要扩容时），这些内层vector在内存中的位置并不连续。

因此，当你访问一个二维向量时，如 vec[i][j]，首先通过外层vector的索引 i 找到对应的内层vector，这个过程是通过连续的指针或迭代器完成的，速度很快。然后，在内层vector中通过索引 j 访问元素，这也是快速的，因为内层vector保证了元素的连续性。但是，从一个内层vector跳转到另一个内层vector时，就可能涉及到非连续的内存访问。

如果你需要一个元素在内存中完全连续的二维数组，可以考虑使用单层数组（一维 vector）并手动计算索引来模拟二维访问。
